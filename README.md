# LMS Hackathon Project

## 🚀 Features
- Course management
- Student dashboard
- Authentication system

## 🛠 Tech Stack
- Laravel (Backend)
- React.js (Frontend)

## 💡 Hackathon Improvements
- AI-based quiz generator (planned)
- Analytics dashboard (planned)

## ⚙️ Setup
1. Clone repo
2. Run `composer install`
3. Run `npm install`
4. Start server
